--- title: "Sprotty" type: sprotty ---

Create diagram visualization tools in your web application with Sprotty

Who: Sprotty is an open-source project developed by a community of developers who are passionate about creating powerful diagramming tools and graphical editors using web technologies.

What: Sprotty provides a framework for rendering diagrams using the Scalable Vector Graphics (SVG) format, with a powerful graph layout engine based on the Eclipse Layout Kernel (ELK) algorithm. It also provides a set of standard components for creating common types of diagrams such as flowcharts, UML diagrams, and state machines, as well as APIs for customizing the layout and behavior of the diagrams.

Where: Sprotty can be used in any web application that supports web technologies such as HTML, CSS, and JavaScript. It can be integrated with various web frameworks such as React, Angular, and Vue.js.

When: Sprotty was first released in 2017 and has since been continuously developed and improved by the community of contributors.

Why: Sprotty makes it easy to create beautiful, interactive diagrams in web applications, with the ability to customize and integrate with external tools and services. It provides a flexible and extensible framework that can be used for a wide range of diagramming needs, from simple flowcharts to advanced graphical editors that integrate with external services. With Sprotty, developers can save time and effort in creating powerful diagramming tools for their applications.
