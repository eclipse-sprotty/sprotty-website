Diagram Vizualization Tools
for your web applications

Sprotty is an open-source project developed by a community of developers
who are passionate about creating powerful diagramming tools and graphical editors
using web technologies.

Get started
Learn more

Sprotty provides a framework for rendering diagrams using the Scalable Vector Graphics (SVG) format, with a powerful graph layout engine based on the Eclipse Layout Kernel (ELK) algorithm. It also provides a set of standard components for creating common types of diagrams such as flowcharts, UML diagrams and state machines, as well as APIs for customizing the layout and behavior of the diagrams.

Customizable and beautiful

Sprotty makes it easy to create beautiful, interactive diagrams in web applications, with the ability to customize and integrate with external tools and services.

From simple to advanced

It provides a flexible and extensible framework that can be used for a wide range of diagramming needs, from simple flowcharts to advanced graphical editors that integrate with external services.

Powerful timesaver

With Sprotty, developers can save time and effort in creating powerful diagramming tools for their applications.

At home in any kind of water...

Sprotty can be used in any web application that supports web technologies such as HTML, CSS, and JavaScript.

Quick to hook...

Sprotty can be integrated with various web frameworks such as React, Angular and Vue.js.

Sprotty is well suited for client-side diagramming, involving no server-side work whatsoever.

This pairs nicely with external applications, allowing you to provide rich visualizations; such as for DSLs.

Sprotty was first released in 2017 and has since been continuously developed and improved by the community of contributors.
